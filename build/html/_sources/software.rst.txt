Software
********

BackProjection and Matched-Filtering (BPMF)
-------------------------------------------

Python package for full earthquake detection and location workflow. :py:data:`BPMF` can be
downloaded at `https://github.com/ebeauce/Seismic_BPMF
<https://github.com/ebeauce/Seismic_BPMF>`_ and the documentation is available at
`https://ebeauce.github.io/Seismic_BPMF/index.html
<https://ebeauce.github.io/Seismic_BPMF/index.html>`_. :py:data:`BPMF` uses :py:data:`beampower` and
:py:data:`fastmatchedfilter`.

Reference: an article is coming soon.

BeamPower
---------

Python package with wrapped C and CUDA-C routines for backprojection.
:py:data:`beampower` can be downloaded at `https://github.com/ebeauce/beampower <https://github.com/ebeauce/beampower>`_ and the documentation is available at `https://ebeauce.github.io/beampower/ <https://ebeauce.github.io/beampower/>`_.

Reference: an article is coming soon.

Fast Matched Filter (FMF)
-------------------------

Python package with wrapped C and CUDA-C routines for template matching.
:py:data:`fastmatchedfilter` can be downloaded at
`https://github.com/beridel/fast_matched_filter
<https://github.com/beridel/fast_matched_filter>`_ and the documentation is
available at `https://ebeauce.github.io/FMF_documentation/
<https://ebeauce.github.io/FMF_documentation/>`_.

Reference: `https://doi.org/10.1785/0220170181
<https://doi.org/10.1785/0220170181>`_.

Iterative Linear Stress Inversion (ILSI)
----------------------------------------

Python package for stress tensor inversion. :py:data:`ILSI` can be downloaded at
`https://github.com/ebeauce/ILSI <https://github.com/ebeauce/ILSI>`_ and the
documentation is available at `https://ebeauce.github.io/ILSI/
<https://ebeauce.github.io/ILSI/>`_.

Reference: `https://doi.org/10.1785/0120210319
<https://doi.org/10.1785/0120210319>`_.
